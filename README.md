# Marina di Pisa Dashboard

Interactive 3D map built with **NiceGUI**, **MapLibre GL**, and **Deck.gl**  
to visualize buildings, road networks, traffic, and noise data.

## Run locally
```bash
python -m venv .venv
source .venv/bin/activate    # (Windows: .venv\Scripts\activate)
pip install -r requirements.txt
python main.py
